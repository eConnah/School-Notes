Selection is implemented using if, then or else statements. Some programming languages also have a switch statement which would count.

Sudocode example:
``` csharp
switch (grade)
{
	case >= 70:
		return 'A';
	case >= 60:
		return 'B';
	case >= 50
		return 'C';
	case >= 40
		return 'D';
	default
		return 'F';
}
```
